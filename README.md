# LabJSON

Run a LabVIEW VI, by taking input as JSON and then return the output as JSON.

# Compatibility
LabVIEW >= 2019

# Install
Install the LabJSON package from [github releases](https://github.com/KarthikAbiram/LabJSON/releases)

Once installed, LabJSON APIs and example can be found under **Functions Palette -> Addons -> LabJSON**
